2. Besondere Hinweise für Kinder und Eltern

📚 Kindgerechte Erklärung:

Liebe Kinder, liebe Eltern! Diese App wurde speziell für Kinder entwickelt. Alle eure Spielstände und Einstellungen werden nur auf eurem eigenen Gerät gespeichert - wie in einem persönlichen Tagebuch, das nur ihr lesen könnt. Wir können eure Daten nicht sehen oder weitergeben.

Altershinweis: Gemäß Art. 8 DSGVO ist für Kinder unter 16 Jahren die Einwilligung der Erziehungsberechtigten erforderlich. Diese App sammelt jedoch keine personenbezogenen Daten, die auf externe Server übertragen werden.

3. Welche Daten werden erfasst?

Lokale Speicherung (localStorage)

Die App speichert folgende Daten ausschließlich lokal in Ihrem Browser:

  • Spielername: Optional eingegebener Name für personalisierte Nachrichten
  • Spielstände: Erreichte Punkte und Spielstatistiken
  • Einstellungen: Schwierigkeitsgrad, Spielzeit, API-Einstellungen
  • Gespeicherte GIFs: Links zu favorisierten Belohnungs-GIFs

Externe Dienste

KLIPY GIF API: Zum Laden von Belohnungs-GIFs wird die KLIPY API verwendet. Dabei werden keine personenbezogenen Daten übertragen, sondern nur anonyme Suchanfragen gestellt.

Attribution: Gemäß KLIPY-Migrationshinweisen wird der Hinweis "powered by KLIPY" in der GIF-Suche und im GIF-Inhaltsbereich angezeigt.

Google AdSense: Diese Website nutzt Google AdSense für die Anzeige von Werbung. Für Kinder sind alle tracking-basierten Werbeanzeigen deaktiviert (Child-Directed Treatment).

4. Wie werden Ihre Daten geschützt?

🛡️ Datenschutz durch Design:

  • Keine Server-Speicherung: Alle Daten bleiben auf Ihrem Gerät
  • Keine Benutzerkonten: Keine Registrierung oder Anmeldung erforderlich
  • Keine Tracking-Cookies: Nur technisch notwendige lokale Speicherung
  • Kindersichere Werbung: Deaktiviertes Tracking für Minderjährige

5. Ihre Rechte

Auskunft, Löschung, Berichtigung

Da alle Daten nur lokal gespeichert werden, können Sie:

  • Daten einsehen: Alle gespeicherten Daten sind in den Einstellungen sichtbar
  • Daten löschen: Über die "Highscore Liste löschen" Funktion oder Browser-Einstellungen
  • Daten ändern: Jederzeit über die App-Einstellungen

Widerspruch gegen Datenverarbeitung

Sie können der Verarbeitung Ihrer Daten jederzeit widersprechen, indem Sie die App nicht mehr nutzen oder die Browser-Daten löschen.

6. Externe Dienste im Detail

Google AdSense

Diese Website nutzt Google AdSense zur Anzeige von Werbung. Für Kinder unter 16 Jahren ist das "Tag for Child Directed Treatment" aktiviert, wodurch:

  • Keine interessensbasierte Werbung angezeigt wird
  • Kein Remarketing stattfindet
  • Keine Cookies für Werbezwecke gesetzt werden

KLIPY GIF API

Zum Laden von Belohnungs-GIFs wird die KLIPY API verwendet. Dabei werden:

  • Keine personenbezogenen Daten übertragen
  • Nur anonyme Suchanfragen gestellt
  • GIFs lokal zwischengespeichert für Offline-Nutzung
  • Die Anbieter-Attribution "powered by KLIPY" sichtbar eingeblendet
